# OOP-Inheritance-Java

Human Management Project

This is a small Java project to practice inheritance.
It has three classes: Human, Assistant, and Guide.

Classes:
Human: base class with name and age.
Assistant: extends Human, adds salary.
Guide: extends Assistant, adds department.

Example Output:

Human{name='Ataa', age=20}

Assistant{name=Ataa, age=25, salary=1000}

Guide{name=Kamal, age=30, salary=3000, department='IT'}
